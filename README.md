# Metronomo

Para regular e marcar o ritmo, o andamento musical.

## Stack

[Docker](https://www.docker.com/)

## Como usar

Instale o [Docker](https://www.docker.com/)

Baixe o projeto

```
bash env/pull.sh
```

Execute o projeto

```
bash env/up.sh
```

Abra o navegador em `localhost`
